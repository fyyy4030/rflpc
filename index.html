rflpc

A low level library for using NXP's LPC17xx SoC. Config is given for MBED prototyping board

Documentation

The documentation of the library is available here. It is generated by Doxygen and corresponds to documentation of the code in the last released version. If you want a more up-to-date documentation (i.e. for the head of the master branch), type 
make doc
at the root of your rflpc repository. You must have Doxygen installed though.
